What's the Weather Like?

How weather changes as you get closer to the equator?
Click Here to View More on GitHub


    Proposal

    The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, I first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

    After assembling the dataset, I used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.


    Process

    First, I pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.
    To accomplish this, I created a loop to generate random latitude and longitude, store them into a data frame for further use.



    Next, it would be the API call.
    Target the OpenWeatherMap url, find the cities weather information which match the latitude and longitude we generated at previous step. Then set the variables like temperature, humidity, cloudiness, wind speed.



    Print out the processing message, and write the data into the data frame we created at step 1. The following diagram will show the first 10 rows of results after the API calls.



    After assembling the dataset, I used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included: temperature, cloudiness, wind speed, and humidity.

    The project side provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.


    Findings



    As expected, the weather becomes significantly warmer as one approaches the equator (0 Deg. Latitude). More interestingly, however, is the fact that he northern hemisphere tends to be warmer this time of year than the southern hemisphere. This may be due to the tilt of the earth at the time of the yar this data was gathered.





    What causes the sky to be cloudy?

    If skies are clear, more heat reaches the earth's surface. However, if skies are cloudy, some of the sun's rays are reflected off the cloud droplets back into space. Therefore, less of the sun's energy is able to reach the earth's surface, which causes the earth to heat up more slowly.

    However, according to the diagram above, it looks like there are no correlations between the latitude and cloudiness.



    One interesting finding is when on the equator, it's much more humidity than other two hemispheres.

    The northern hemisphere tends to be more humidity this time of year than the southern hemisphere, especially around N25° and N75°.



    As the diagram above, one fact we can easyily figure out: the higher the dimension, the faster the wind speed.
